CI-CD process

  1. Create a new repository using Github Desktop (In my case gitkracken)
  2. Publish your repo as prvate or public
  3. create an index.html (very important) with some content
  4. commit and push your changes using github desktop (Gitkracken for me)
  5. Go to netlify
  6. Add new site from git
  7. Profit and change the site name
